**스윙스(Base ball shop)**
=============
스윙스는 JAVA와 Srping MVC를 기반으로 제작한 프로젝트입니다.

야구 용품판매와 커뮤니티 내의 자유게시판,야구소식,경기일정을 통해 

단순한 야구용품 판매만을 위한 사이트가 아닌

야구와 관련된 정보를 얻고 주고받으며 소통하는 사이트를 추구하였습니다.
